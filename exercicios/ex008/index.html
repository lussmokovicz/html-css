Não semântica:

Nessa frase temos o termo em negrito usando a tag <b>

Semântica:

Nesta frase aqui, tenho um termo em destaque usando a tag <strong>

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Mas se tu quiser mudar a cor do marcador, precisa usar CSS, então fica assim ó:
Dentro da tag <mark> você coloca style=" backgound color: nome da cor; "

Algo assim: Lorem, ipsum dolor sit amet consectetur adipisicing elit. Quod, magnam saepe repellat quibusdam sit mollitia illo dolor pariatur ad amet, nihil doloribus, voluptatibus reprehenderit unde rerum ratione corrupti accusantium id.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Use <del>

Texto inscrito

Podemos marcar um texto como inscrito para dar uma ênfase e indicar que ele foi inserido depois. Use <ins>

Existe também o sublinhado, com a tag <u> que é não semântica, que tá prestes a morrer :(

Texto sobrescrito

Para inserir coisas do tipo x20+3 (x a 20 mais 3), use <sup>

Texto subescrito

Para inserir coisas do tipo H2O (fórmulas moleculares, por exemplo), use <sub>
